# Quiz Portal Template

A modern, responsive quiz website template with white + light blue theme.

## Features

- ✅ Fully Responsive Design
- ✅ Modern White + Light Blue Theme
- ✅ Bootstrap 5 Framework
- ✅ Bootstrap Icons
- ✅ Smooth Animations
- ✅ Clean and Professional Design
- ✅ Multiple Pages Included

## Pages Included

1. **index.html** - Homepage with hero section, features, statistics, and more
2. **about.html** - About Us page
3. **contact.html** - Contact page with form
4. **quizzes.html** - Quizzes listing page with filters
5. **login.html** - User login page with social login options
6. **register.html** - User registration page with validation
7. Additional pages can be created following the same structure

## File Structure

```
quize-template/
├── index.html          # Homepage
├── about.html          # About page
├── contact.html        # Contact page
├── quizzes.html        # Quizzes listing page
├── login.html          # Login page
├── register.html       # Registration page
├── css/
│   ├── style.css       # Main stylesheet
│   ├── pages.css       # Pages specific styles
│   └── auth.css        # Authentication pages styles
├── js/
│   ├── main.js         # Main JavaScript
│   └── auth.js         # Authentication JavaScript
└── images/             # Image assets folder
```

## Color Scheme

- Primary Blue: #4A90E2
- Light Blue: #E3F2FD
- Dark Blue: #1976D2
- White: #FFFFFF
- Light Gray: #F5F5F5

## Usage

1. Open `index.html` in a web browser
2. Customize content, colors, and styles as needed
3. Add your own images to the `images/` folder
4. Modify CSS files to match your brand
